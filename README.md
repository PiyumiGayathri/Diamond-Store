# Diamond Store - Buy & Sell Exclusive Gemstones

The Diamond Store web application provides a platform for users to buy and sell gemstones. It 
includes separate interfaces and functionalities for users and administrators. Users can perform 
actions such as signing up, signing in, purchasing items, managing their cart and wish list, 
providing feedback, and maintaining their profiles. After sign-in to the system and sell the
products. Administrators have additional capabilities includes managing products, managing 
users, and managing sales history details.
